# Power BI Project-Sales Dashboard
### Project Goal
Create a simple yet powerful Sales Dashboard in Power BI showing total sales, quantity sold, performance by category, city, region, and month.
•	Clean & transform data (Power Query)
•	Build a basic data model
•	Create DAX measures
•	Design visuals (cards, charts, map)
•	Publish your report(optional)
### Power Query Cleaning steps performed
Remove Duplicates
Trim and Clean Text Columns
Change Data Types
Replace Missing Values
Fix Incorrect Shipping Dates
Remove Unnecessary Columns
